# 🛍️ WHOPPER MARKET

## Функционал
- Клиент: покупка товаров, корзина, заказы, QR-код
- Продавец: добавление товаров, управление
- ПВЗ: приём и выдача заказов, сканер QR

## Установка
1. Скачайте все файлы
2. Откройте `index.html` в браузере

## Технологии
- Firebase (Database + Storage)
- QRCode.js
- jsQR

## Доступы
- Клиент: регистрация по телефону
- Продавец: регистрация через форму
- ПВЗ: номер и пароль (создаются вручную в Firebase)
