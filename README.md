# Маркетплейс с Firebase

## 🚀 Функционал

### 4 роли:
- **Клиент** - покупает товары, оформляет заказы, QR-код
- **Продавец** - выкладывает товары
- **ПВЗ** - выдает заказы по QR-коду
- **Админ** - управляет ПВЗ

### 🔑 Доступы:
- Админ: логин `2347`, пароль `2203`
- ПВЗ: создает админ
- Продавец/Клиент: регистрация через email

## 📁 Установка

1. Скопируйте все файлы на сервер
2. Откройте `vxod.html`

## 📱 Приложение

Для установки на телефон:
- Android/iPhone: откройте в браузере, нажмите "Добавить на экран домой"

## 🛠 Технологии

- Firebase (Auth + Realtime Database)
- QRCode.js
- PWA
